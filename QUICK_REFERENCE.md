# 快速参考 - qualityFirst 参数

## 🎯 一句话总结

添加 `true` 参数到脚本末尾，即可启用**质量优先模式**，获得极致肤色还原、毛发细节清晰、嘴唇红润的效果。

---

## 📋 快速命令

### GIF - 质量优先
```bash
./video2gif.sh "input.mp4" "output.gif" 00:00:00 00:00:10 30 1080 true
```

### APNG - 质量优先
```bash
./video2apng.sh "input.mp4" "output.apng" 00:00:00 00:00:10 30 1080 3 true
```

### WebP - 质量优先
```bash
./video2webp_stand.sh "input.mp4" "output.webp" 00:00:00 00:00:10 30 1080 95 true
```

## ✅ 何时使用

### 使用平衡模式（默认）
- 网页展示
- 移动端
- 快速处理

### 使用质量优先模式
- 人像特写
- 高端展示
- 肤色细节重要
- 毛发需要清晰

---

## 🎨 参数变化

### 降噪（Denoise）
- 平衡：0.3:0.2:0.8:0.6
- 质量：0.2:0.1:0.5:0.3 ← **更少降噪，保留细节**

### 锐化（Sharpen）
- 平衡：5:5:0.8:3:3:0.4
- 质量：7:7:1.2:4:4:0.6 ← **强度提升50%**

### 饱和度（Saturation）
- 平衡：1.05-1.08
- 质量：1.12 ← **提升7%，肤色更饱满**

### GIF 阈值（Threshold）
- 平衡：90
- 质量：95 ← **保留更多色彩**

## 📞 常见问题

**Q: 质量优先模式会慢多少？**
A: 约 7-12% 更慢，但质量提升 25-30%

**Q: 文件会大多少？**
A: 约 20-40% 更大，但细节完美

**Q: 哪个格式最适合质量优先？**
A: APNG（24位真彩）> GIF（256色）> WebP（压缩优先）

**Q: 可以同时调整其他参数吗？**
A: 可以，qualityFirst 只是预设，其他参数仍可自定义
