## 🔧 参数调整逻辑

### 平衡模式（默认）
```bash
# 降噪：轻度
hqdn3d=0.3:0.2:0.8:0.6

# 锐化：标准
unsharp=5:5:0.8:3:3:0.4

# 色彩：适度
eq=contrast=1.02:brightness=0.01:saturation=1.05-1.08

# 压缩：优先
GIF阈值=90, APNG压缩=5, WebP质量=90
```

### 质量优先模式
```bash
# 降噪：最小
hqdn3d=0.2:0.1:0.5:0.3

# 锐化：强化（+50%）
unsharp=7:7:1.2:4:4:0.6

# 色彩：增强（+7%饱和度）
eq=contrast=1.03:brightness=0.01:saturation=1.12

# 压缩：质量优先
GIF阈值=95, APNG压缩=3, WebP质量=95
```
